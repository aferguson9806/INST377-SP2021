Python: Features, and Strengths

Python is likely a programming language that many UMD InfoSci students are familiar with, as this is the language taught in both the introductory programming class (INST126) and the object oriented programming class (INST326) in the iSchool

One of the key features of Python is its code block syntax, which throws off lots of programmers who are used to other languages. Python uses indentation, or "whitespace" to specify code that belongs to a block. Python is very strict about whitespace as if something is on the wrong indentation level, the Python interpreter will read your code the wrong way and you won't get the result you expect.

This form of whitespace specification is also known as "implicit" code block syntax, as there isn't a specific character or keyword used to specify a block of code.

In terms of graphic user interface design, python generally relies on a library known as tkinter to build GUIs.

In my opinion, this is not where Python thrives. Python is very strong when it comes to data science. Similar to tkinter, Python has a library called Pandas which is famous for its ability to create "dataframes" Dataframes are like excel spreadsheets in that there are columns and rows and each cell has a significance in terms of the data.

Pandas is very useful when analyzing large datasets. It is very easy to turn a csv file or an excel file into a pandas dataframe and do work on it in Python. It is also very easy to take a Python data structure like a list of lists or a dictionary and turn it into a pandas dataframe.
